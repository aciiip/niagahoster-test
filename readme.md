#Niagahoster Test
Niagahoster landing page by Nasib.

## System Requirements
- PHP >= 7.1.3
- BCMath PHP Extension
- Ctype PHP Extension
- JSON PHP Extension
- Mbstring PHP Extension
- OpenSSL PHP Extension
- PDO PHP Extension
- Tokenizer PHP Extension
- XML PHP Extension

## Usage
### Local Development Server
`php artisan serve`

### Data Storage
`/public/data.json`
